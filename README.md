## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/venkateshtata/net_handle.git
   cd net-handle
   ```
2. Install requirements:
    ```bash
    pip install -r requirements.txt
    ```
3. Start the workflow API
    ```bash
    uvicorn main:app
    ```
4. Start the Interface
    ```bash
    cd frontend
    npm start
    ```
